AJAX
Asynchronous JavaScript and XML
자바스크립트와 XML을 이용한 비동기통신
xml : eXtensible Markup Language
-> 문법 HTML
<태그의 이름 속성이름=속성값></이름>
HTML :: 이미 정의되어있는 태그이름
XML :: 태그 이름을 정의

<person>
	<attr>
		<name>홍길동</name>
		<age>30</age>
	</attr>
	<datas>
		<korean>80</korean>
		<eng>70</eng>
		<math>60</math>
	<datas>
</person>
-------------

    'idx' : 'person',
    'attr' : [{ 'name' : '홍길동'}, {'age':30} ],
    'datas' : [ {'korean':80},{'eng':70},{'math':60}]
